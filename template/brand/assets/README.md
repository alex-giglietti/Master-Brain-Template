# Brand Assets

This directory contains all visual brand elements.

## Directory Structure

```
assets/
├── logos/          # Logo files in various formats
├── favicon/        # Favicon files for web
├── business-cards/ # Business card designs
└── banners/        # Social media and web banners
```

## Logo Guidelines

### File Formats
- **SVG** - For web (scalable, preferred)
- **PNG** - For web with transparency
- **PDF** - For print materials
- **EPS** - For professional printing

### Versions to Include
- Primary logo (full color)
- Secondary logo (simplified)
- Icon only (for small sizes)
- White version (for dark backgrounds)
- Black version (for light backgrounds)

## Favicon Requirements

Include these sizes:
- `favicon.ico` (16x16, 32x32, 48x48)
- `apple-touch-icon.png` (180x180)
- `favicon-32x32.png`
- `favicon-16x16.png`
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`

## Business Cards

Include:
- Print-ready PDF (with bleed)
- Editable source file (AI, PSD, or Figma)
- Preview image

## Banners

### Standard Sizes
| Platform | Size |
|----------|------|
| LinkedIn Cover | 1584 x 396 px |
| Twitter Header | 1500 x 500 px |
| Facebook Cover | 820 x 312 px |
| YouTube Banner | 2560 x 1440 px |
| Email Header | 600 x 200 px |

## Naming Convention

Use this naming pattern:
```
[company]-[asset-type]-[variant]-[size].[format]

Examples:
- acme-logo-primary-full.svg
- acme-logo-icon-white.png
- acme-banner-linkedin-2024.png
```
