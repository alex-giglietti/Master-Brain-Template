# Convert Playbooks

Sales qualification, objection handling, and closing strategies.

## Purpose

The Convert stage turns interested prospects into paying customers. Playbooks in this folder help navigate sales conversations effectively.

## Key Playbooks to Create

### 1. `lead-qualification.md`
Criteria and questions to qualify leads.
- Budget questions
- Authority verification
- Need assessment
- Timeline discovery

### 2. `discovery-call.md`
Structure for initial sales calls.
- Opening script
- Question flow
- Active listening prompts
- Next step transitions

### 3. `objections.md`
Common objections and how to handle them.
- Price objections
- Timing objections
- Competitor comparisons
- Trust concerns

### 4. `proposal-presentation.md`
How to present and walk through proposals.
- Structure
- Key talking points
- Handling questions

### 5. `closing-techniques.md`
Strategies for closing deals.
- Trial closes
- Urgency creation
- Final ask scripts

## Bot Behavior in Convert Stage

When helping with sales conversations, the bot should:

1. **Listen first** - Understand before pitching
2. **Match solutions to problems** - Connect offers to stated needs
3. **Handle objections gracefully** - Use approved responses
4. **Know when to escalate** - Complex negotiations need humans
5. **Follow up persistently** - Multiple touches often needed

## Common Objections Quick Reference

| Objection | Response Type |
|-----------|---------------|
| "Too expensive" | Value justification |
| "Need to think about it" | Urgency + follow-up |
| "Already have a solution" | Differentiation |
| "Not the right time" | Future booking |
| "Need to talk to partner/team" | Offer to help convince |
