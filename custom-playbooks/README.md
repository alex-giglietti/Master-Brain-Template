# Custom Playbooks

This folder is **yours**. Add any custom playbooks, SOPs, or guides here.

Your AI will read these alongside the official playbooks. Updates to the main brain will **never** overwrite or remove files in this folder.

## How to Add a Custom Playbook

1. Create a `.md` file in this folder
2. Name it descriptively: `my-email-sequence-playbook.md`
3. Your AI will automatically find and use it

```markdown

<!-- When should AI reference this? -->

## Steps
1. Step one...
2. Step two...

<!-- Any templates or examples -->
```
